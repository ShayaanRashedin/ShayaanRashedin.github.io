Cybersecurity Home Lab

Home Lab Overview

To build real-world cybersecurity skills, I created a home lab using Kali Linux and Metasploitable 2 to explore both offensive (red team) and defensive (blue team) techniques. This environment allows me to practice vulnerability scanning, exploitation, web application attacks, log analysis, and intrusion detection using industry-standard tools like Metasploit, Nmap, Burp Suite, Wireshark, ELK, and Security Onion.

Each project is documented to show not only how attacks work, but also how to detect and prevent them—demonstrating a full-stack approach to cybersecurity and my growth as a practitioner.
