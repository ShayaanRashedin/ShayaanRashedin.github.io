My Cybersecurity Home Lab

My home lab includes Kali Linux, Metasploitable2, and a pfSense firewall VM running on a spare laptop. I've used this setup to simulate penetration testing scenarios, audit vulnerable services, and log attack traces.

Key components:

  • Kali Linux (latest rolling release)
  • Metasploitable2 vulnerable VM
  • pfSense firewall with custom rules
  • ELK stack for log aggregation and visualization
  • VirtualBox host on Ubuntu server

Feel free to explore my GitHub repo for full setup scripts and diagrams.
