Cybersecurity Home Lab

I built a home lab to practice both offensive and defensive skills using Kali Linux (attacker), Metasploitable 2 (intentionally vulnerable target), and an Ubuntu VM running Splunk as the SIEM, all within isolated VirtualBox networks. The lab supports vulnerability scanning (OpenVAS/GVM, Nmap), exploitation and post-exploitation (Metasploit), packet capture/analysis, and blue-team detection/response workflows with Splunk dashboards and alerts.
