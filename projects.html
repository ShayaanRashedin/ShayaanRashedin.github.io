Using Linux Commands to Manage File Permissions

Project Description

I managed file and directory permissions for a research team: audited existing settings, compared against policy, and corrected discrepancies using Linux permission commands.

Current File Permissions

The /home/researcher2/projects directory contains:

  • project_k.txt – rw-rw-rw-
  • .project_x.txt – rw-r-----

Check Permissions

ls -la /home/researcher2/projects

Describe the String

-rw-rw-r-- 1 researcher2 researcher2 1234 project_k.txt

This breaks down as: -=file, rw-=owner, rw-=group, r--=others.

Change Permissions

chmod o-w /home/researcher2/projects/project_k.txt

Result

-rw-rw-r-- 1 researcher2 researcher2 1234 project_k.txt

Summary

Used chmod to enforce policy: removed “write” from others and secured hidden files and directories.

Applying Filters to SQL Queries

Project Description

I leveraged SQL filtering (AND, OR, NOT, LIKE) on login and employee tables to pinpoint anomalies and support security investigations.

After-Hours Failed Logins

SELECT * FROM log_in_attempts
WHERE login_time > '18:00'
  AND success = FALSE;

Specific-Date Logins

SELECT * FROM log_in_attempts
WHERE login_date IN ('2022-05-08','2022-05-09');

Outside Mexico

SELECT * FROM log_in_attempts
WHERE country NOT LIKE 'MEX%';

Marketing Employees

SELECT * FROM employees
WHERE department = 'Marketing'
  AND office LIKE 'East%';

Finance or Sales

SELECT * FROM employees
WHERE department IN ('Finance','Sales');

Not in IT

SELECT * FROM employees
WHERE department != 'Information Technology';

Summary

Demonstrated proficiency in constructing parameterized SQL filters to extract relevant security-related data for analysis.

Importing and Parsing a Text File in a Security-Related Scenario

Project Description

Automated the update of an IP allow-list in allow_list.txt by removing revoked addresses from a remove_list using Python.

Read File

with open("allow_list.txt","r") as f:
    ips = f.read().split()

Filter

for ip in remove_list:
    if ip in ips: ips.remove(ip)

Write Back

with open("allow_list.txt","w") as f:
    f.write("\n".join(ips))

Summary

Ensured only authorized IPs remained on the allow-list, strengthening network access controls.
