Portfolio Projects

Conducting a Security Audit

This security audit project involved assessing an organization’s external-facing infrastructure. I scanned for open ports, identified outdated software versions, and provided remediation recommendations. Sample findings: outdated SSH configuration, missing security patches on web server, and weak TLS ciphers.

Analyzing Network Structure and Security

In this project, I mapped out the network topology using NetBox, then used Nmap to discover live hosts and trace routes. I documented VLAN segmentation, firewall rule interactions, and highlighted potential lateral movement paths through misconfigured ACLs.

Using Linux Commands to Manage File Permissions

I scripted a set of Bash commands to audit and correct file permissions across a Linux file server. The script checks for world-writable files, enforces minimum permission standards, and logs any changes to a centralized report for review.

Applying Filters to SQL Queries

This exercise demonstrated how to write parameterized SQL queries to extract only relevant data from a customer database. I applied WHERE clauses, JOIN filters, and used ORDER BY to sort results efficiently, improving query performance by 30% on large datasets.

Identifying Vulnerabilities for a Small Business

For a mock SMB, I ran vulnerability scans with Nessus, categorized findings by severity, and drafted an executive summary. Key issues included default credentials on IoT devices, outdated CMS plugins, and absence of multi-factor authentication for admin accounts.

Documenting Incidents with an Incident Handler's Journal

This journal entry details a simulated phishing incident. It covers detection methods, containment steps, eradication procedures, and post-incident lessons learned. I logged timelines, stakeholders involved, and updated playbooks accordingly.

Importing and Parsing a Text File in a Security-Related Scenario

I wrote a Python utility to ingest log files exported from a SIEM, parse out key indicators of compromise, and reformat the data into CSV for further analysis. The script handles edge cases like truncated lines and missing timestamps.
