Professional Statement

I am a third year Computer Engineering student at Toronto Metropolitan University with a strong foundation in both software and hardware. My background includes object oriented design and data structures, digital systems such as finite state machines, ALU design, and register level logic, and electronic circuits. I apply these fundamentals to hands on security projects in a three virtual machine home lab that includes Kali as the attacker, Metasploitable 2 as the intentionally vulnerable target, and Ubuntu with Splunk as the security monitoring and analysis layer. My work spans network reconnaissance using ICMP and Nmap, vulnerability assessment using OpenVAS and GVM, exploit proof of concepts using Metasploit, and blue team analytics in Splunk where I build searches, dashboards, and alerts. I document methodology, evidence, and remediation steps to demonstrate engineering discipline, clarity, and repeatability.
